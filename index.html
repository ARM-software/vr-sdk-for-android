VR SDK for Android 0.1.1 Documentation

Welcome to VR SDK 0.1.1.

About

The VR SDK for Android is a collection of resources to help you build VR applications for a platform with a Mali GPU and an ARM processor. You can use it for creating new applications, training, and exploration of implementation possibilities.

You can use the VR SDK to produce applications that will run on any devices that run Android™ on an ARM processor.

Requirements

To build and run the VR sample applications you will need:

  • An ARM based development board with a Mali series GPU running Android.
  • The Latest Android Studio.
  • The Latest Android NDK ( can be installed from Android Studio ).
